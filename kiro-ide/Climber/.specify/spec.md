# Climber — Specification

## Overview
Climber is an AI-powered climbing companion app. Users can log climbs, track progress, and receive Gemini-powered route recommendations.

## Goals
- Help climbers log and review their climbing sessions
- Provide AI-driven insights on performance and improvement areas
- Suggest routes based on skill level and goals

## Features

### 1. Climb Logger
- Log a climb with: route name, grade, location, date, result (sent/attempt), notes
- View history of past climbs

### 2. Progress Dashboard
- Visual charts of grades climbed over time
- Success rate per grade
- Personal bests

### 3. AI Route Suggestions (Gemini)
- Input: current max grade, preferred style (bouldering/sport/trad)
- Output: suggested next routes to try with reasoning

### 4. Profile
- Climber name, home gym, climbing since date
- Goals (e.g. "Send V6 by June")

## Out of Scope (v1)
- Social/sharing features
- Real-time route databases
- Offline mode
