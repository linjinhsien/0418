# Climber — Project Constitution

## Vision
A climbing companion app built with AI Studio (Gemini), helping climbers track routes, progress, and goals.

## Core Principles
- **AI-first**: Leverage Gemini for intelligent route suggestions and progress analysis
- **Simple UX**: Fast, intuitive interface focused on the climber's workflow
- **Data integrity**: Accurate tracking of climbs, grades, and performance over time

## Development Guidelines
- Keep components small and focused
- Prioritize mobile-first responsive design
- Use Gemini API for any AI-powered features
- All features must be testable and documented

## Quality Standards
- Code must be readable and well-commented
- UI must be accessible (WCAG 2.1 AA)
- Performance: initial load under 3s
